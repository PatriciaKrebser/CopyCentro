## Estilo y Personalidad de Marca

Este sistema de diseño personifica la energía creativa de una librería moderna y un centro de impresión dinámico. Se aleja de la rigidez corporativa para abrazar una estética **Alegre, Cálida y Profesional**. El objetivo es evocar la sensación de un lienzo en blanco listo para ser llenado, utilizando una base limpia con explosiones de color que guían al usuario.

El estilo visual combina el **Minimalismo Moderno** con toques de **High-Contrast**, priorizando la legibilidad y la facilidad de navegación para estudiantes, profesionales y creativos. La interfaz debe sentirse luminosa y espaciosa, como una tienda bien organizada y llena de luz natural.

## Colores

La paleta se fundamenta en el modelo CMYK del logo, pero optimizada para pantallas con tonos vibrantes y sus variantes pasteles para fondos.

- **Cian Primario (#00AEEF):** Utilizado para acciones principales, enlaces y elementos de navegación. Transmite confianza y frescura tecnológica.
- **Magenta Secundario (#EC008C):** Color de acento para promociones, notificaciones y elementos que requieren atención creativa.
- **Amarillo Energía (#FFF200):** Se usa con moderación para resaltar ofertas o etiquetas, siempre manteniendo el contraste con el texto oscuro.
- **Negro Profundo (#231F20):** Reservado para la tipografía principal y logotipos, asegurando una legibilidad impecable.
- **Fondos de Superficie:** Se deben utilizar versiones al 5-10% de opacidad de los colores primarios para diferenciar secciones sin saturar la vista.

## Tipografía

Se ha seleccionado **Plus Jakarta Sans** por su naturaleza geométrica pero amable. Sus terminaciones redondeadas y su amplia altura de x la hacen extremadamente legible tanto en cartelería digital como en interfaces móviles.

Los titulares deben usar pesos pesados (Bold/ExtraBold) para inyectar energía y jerarquía clara. El cuerpo de texto mantiene un peso regular para facilitar la lectura prolongada en catálogos de productos o descripciones de servicios técnicos.

## Diseño y Espaciado

El sistema utiliza una **retícula fluida de 12 columnas** para escritorio y una retícula de **4 columnas** para dispositivos móviles. El ritmo visual se basa en una unidad base de 8px, garantizando consistencia matemática en todos los componentes.

El espaciado es generoso para evitar la sensación de "desorden de oficina". Se deben priorizar márgenes amplios alrededor de las imágenes de productos para que los colores del material escolar "respiren". Las secciones se dividen mediante espacios en blanco (whitespace) en lugar de líneas divisorias pesadas.

## Elevación y Profundidad

La jerarquía se construye mediante **Capas Tonales** y sombras ambientales suaves. En lugar de sombras negras genéricas, se utilizan sombras con un ligero tinte del color primario o neutro, con una opacidad muy baja (5-10%) y una dispersión amplia.

Los elementos interactivos (tarjetas de producto, modales) se elevan sutilmente sobre el fondo para invitar al clic. No se busca un efecto 3D realista, sino una sensación de capas de papel superpuestas con limpieza.

## Formas

El lenguaje de formas es **Redondeado**, reflejando amabilidad y accesibilidad. 

- **Contenedores y Tarjetas:** Utilizan el radio estándar de `1rem` (16px) para suavizar las esquinas.
- **Botones y Chips:** Siguen una estética orgánica que complementa la tipografía Plus Jakarta Sans.
- **Iconografía:** Debe presentar bordes redondeados y grosores de línea consistentes con el peso de la tipografía de cuerpo (aprox. 1.5px o 2px).

## Componentes

### Botones
- **Primario:** Fondo cian con texto blanco. Efecto de escala suave al presionar.
- **Secundario:** Borde magenta con fondo transparente y texto magenta.
- **Estado:** Bordes muy redondeados, casi tipo píldora para acciones rápidas.

### Tarjetas de Producto
- Fondo blanco puro con una sombra de elevación nivel 1.
- El nombre del producto en `label-md` y el precio destacado en el color secundario (magenta).
- Esquinas con radio de `rounded-lg`.

### Campos de Entrada (Inputs)
- Bordes suaves en gris claro que cambian a cian al recibir el foco.
- Etiquetas (labels) siempre visibles para evitar confusión en formularios de pedidos de impresión.

### Chips y Etiquetas
- Pequeñas píldoras de color pastel (cian suave, amarillo suave) para categorizar productos (ej: "Escolar", "Técnico", "Oferta").

### Listas de Servicios
- Iconos de colores vibrantes a la izquierda, seguidos de un titular claro y una descripción breve, utilizando un espaciado `sm` entre elementos.
