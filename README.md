# CopyCentro

A modern website for a stationery store and print center located in Rosario, Argentina.

> This project is currently under active development as part of my Full Stack Web Development learning journey.

---

## Features

- Responsive landing page
- Modern and clean user interface
- Services section
- Contact information
- WhatsApp integration
- Smooth navigation between sections

---

## Technologies

- HTML5
- Tailwind CSS
- Git
- GitHub
