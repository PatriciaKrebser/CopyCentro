# Métodos de Pago

## Introducción

Este documento contiene la información relacionada con las formas de pago aceptadas por CopyCentro.

El asistente debe utilizar esta información cuando un cliente consulte cómo puede abonar una compra o un servicio.

Si algún método de pago no está confirmado en la base de conocimientos, el asistente nunca deberá asumir que está disponible.

---

# Efectivo

CopyCentro acepta pagos en efectivo.

---

# Tarjetas de débito

Consultar disponibilidad en el local.

El asistente no debe confirmar este medio de pago si no existe información actualizada.

---

# Tarjetas de crédito

Consultar disponibilidad en el local.

El asistente no debe confirmar promociones, cuotas o bancos adheridos si esa información no está disponible.

---

# Mercado Pago

Consultar disponibilidad en el local.

Si existen promociones vigentes, deberán estar documentadas en la base de conocimientos.

---

# Transferencias bancarias

Consultar disponibilidad en el local.

El asistente no debe proporcionar datos bancarios si no forman parte de la información oficial.

---

# Facturación

Si el cliente necesita factura, el asistente deberá indicar que consulte directamente con CopyCentro para conocer los requisitos y el tipo de comprobante disponible.

---

# Consultas sobre promociones

El asistente nunca debe inventar promociones.

Si no existe información actualizada sobre descuentos, cuotas o beneficios, deberá recomendar al cliente consultar directamente con CopyCentro.

---

# Importante

El asistente debe responder únicamente utilizando la información oficial disponible.

No debe asumir que un medio de pago está disponible si no figura en la base de conocimientos.

Ante cualquier duda, deberá recomendar al cliente comunicarse con CopyCentro.
