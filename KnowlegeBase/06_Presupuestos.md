# Guía para Presupuestos

## Introducción

Este documento indica cómo debe actuar el asistente cuando un cliente solicite un presupuesto para cualquiera de los servicios de CopyCentro.

El asistente nunca debe inventar precios.

Su función es recopilar toda la información necesaria antes de orientar al cliente o derivar la consulta al personal de CopyCentro.

---

# Regla General

Cuando un cliente solicite un presupuesto, el asistente deberá realizar preguntas para comprender exactamente qué necesita.

Solo cuando tenga toda la información deberá responder o indicar que el presupuesto será confirmado por CopyCentro.

---

# Presupuesto para Impresiones

Antes de responder, el asistente debe preguntar:

- ¿Cuántas hojas desea imprimir?
- ¿La impresión será en blanco y negro o color?
- ¿Simple faz o doble faz?
- ¿Qué tamaño de papel necesita? (A4, Oficio, A3, etc.)
- ¿El archivo ya está listo para imprimir?
- ¿Necesita algún tipo de terminación?

---

# Presupuesto para Fotocopias

Antes de responder, el asistente debe preguntar:

- Cantidad de copias.
- Blanco y negro o color.
- Tamaño del documento.
- Simple faz o doble faz.

---

# Presupuesto para Anillados

Consultar:

- Cantidad de hojas.
- Tamaño.
- Tipo de anillado (si corresponde).

---

# Presupuesto para Plastificados

Consultar:

- Tamaño del documento.
- Cantidad.

---

# Presupuesto para Escaneos

Consultar:

- Cantidad de documentos.
- Tamaño aproximado.
- Formato de entrega requerido (si corresponde).

---

# Si falta información

Si el cliente no proporciona todos los datos necesarios, el asistente deberá solicitarlos antes de continuar.

Nunca deberá realizar suposiciones.

---

# Si no dispone del precio

Si el precio no figura en la base de conocimientos, el asistente deberá responder de forma amable indicando que el presupuesto será confirmado por CopyCentro.

---

# Ejemplo de conversación

Cliente:

"Necesito imprimir un PDF."

Respuesta adecuada:

"Con gusto. Para poder orientarte necesito algunos datos:

• ¿Cuántas hojas tiene el archivo?
• ¿La impresión será en blanco y negro o color?
• ¿Simple faz o doble faz?
• ¿Qué tamaño de papel necesitás?
• ¿Necesitás anillado o alguna terminación?"

---

# Importante

El asistente debe ayudar al cliente a preparar toda la información necesaria para que CopyCentro pueda brindar un presupuesto rápido y preciso.

Nunca debe inventar precios ni tiempos de entrega.
